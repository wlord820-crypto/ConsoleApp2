# Student Results Processing System

## Project Overview

The Student Results Processing System is a C# console application developed as part of the CE 380 Programming with C# course. The system allows users to enter student information and course scores, process academic results, and generate detailed performance reports.

The application supports multiple students and calculates important academic metrics such as total scores, average scores, grades, pass/fail status, class average, best-performing student, and lowest-performing student.

## Features

* Enter details for any number of students
* Record scores for five courses
* Validate score inputs (0–100)
* Calculate total scores
* Calculate average scores
* Assign grades (A–F)
* Determine pass/fail status
* Display detailed student reports
* Identify the best-performing student
* Identify the student with the lowest average
* Calculate and display the class average
* User-friendly menu-driven interface
